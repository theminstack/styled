map

  • map<TNewInnerProps>(cb: (props: TExtendedProps) => Partial<TInnerProps> & TNewInnerProps): IStyledTemplateMod<TStatic, TOuterProps, TInnerProps, Flat<TNewInnerProps>>

  • Transform all props based on previous props. The new props must still be compatible with the wrapped component's props.

    This method removes props that are not returned.

    • Try the use method if you want to provide default prop values.
    • Try the set method if you want to set (not remove) some prop values.

set

  • set<TNewInnerProps>(cb: (props: TExtendedProps) => Partial<TInnerProps> & TNewInnerProps): IStyledTemplateMod<TStatic, TOuterProps, TInnerProps, Flat<Merge<TExtendedProps, TNewInnerProps>>>

  • Partially set props. This is effectively a "merge" with undefined value skipping.

    Be careful that you do not unconditionally overwrite props passed in to the styled component!

    • Try the use method if you want to provide default prop values (when the current value is undefined).
    • Try the map method if you want to set and remove values.

  • use<TNewInnerProps>(cb: (props: TExtendedProps) => Partial<TInnerProps> & TNewInnerProps): IStyledTemplateMod<TStatic, TOuterProps, TInnerProps, Flat<Defaults<TExtendedProps, TNewInnerProps>>>

  • Use prop default values. As the name implies, this callback is generally where you should use hooks (eg. useTheme()).

    • Try the set method if you want to set prop values regardless of the current value.
    • Try the map method if you want to set and remove values.
