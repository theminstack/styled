Interface IStyledSelector

Values of this type can be used as selectors in tagged template style strings. The two argument overload of the styled function returns styled components which implement this interface.

const Foo = styled('div', 'Foo');
const Bar = styled('div')`
  ${Foo} {
    color: red;
  }
`;
